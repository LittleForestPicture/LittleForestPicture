# 스프링 토픽 발표 스터디

## Description

-   `일주일`에 `한명`씩 정해진 토픽에 대해서 발표를 진행합니다.
-   시간은 자유이지만 원활한 진행을 위해서 `한시간 안쪽`으로 분량을 맞춰주시기 바랍니다.
-   발표를 한 뒤에 `질의응답` 시간을 가집니다. 발표자 이외의 분들은 토픽에 대해서 간단한 `사전공부`를 하시는 것을 권장드립니다.
